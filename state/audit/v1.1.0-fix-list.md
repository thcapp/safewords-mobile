- **Batch A** — Service layer extensions: `GroupRepository` 11 new methods, `DrillService`, `SmsInviteService`, `BiometricService`

- **Batch D** — Settings every row + Plain mode exit + Plain Home CTA + Verify active-group respect
- **Batch E** — Widget timeline fix (P0 #12), QR scanner repair (P0 #11)
